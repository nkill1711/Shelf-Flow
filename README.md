# ShelfFlow

ShelfFlow is a fully responsive web-based inventory management system designed to simplify inventory tracking using barcode technology. It helps businesses manage stock efficiently with real-time updates, product organization, and easy inventory monitoring.

---

## Features

- Inventory Management
-  Barcode-based Product Tracking
- Real-time Stock Updates
-  User Authentication
- Fully Responsive UI
-  Fast and Modern MERN Stack Application
